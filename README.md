# CSV Data Pipeline

Dieses Repository dient als Lernprojekt zur praktischen Anwendung von Python.

## Inhalte

* Arbeiten mit Klassen und Pydantic-Modellen
* Implementierung einfacher Business-Logik
* Grundlagen von FastAPI
* Lesen und Verarbeiten von CSV-Dateien
* Nutzung von Git und GitHub zur Versionsverwaltung

## Ziel

Das Projekt wurde erstellt, um praktische Erfahrungen im Aufbau kleiner Python-Anwendungen zu sammeln und grundlegende Backend-Konzepte zu verstehen.
